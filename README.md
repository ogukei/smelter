# Smelter

Simple web app interface builder in Rust.

## Setup

Install the latest npm using n on Ubuntu 20.04 LTS.

```
sudo apt install nodejs npm
sudo npm install -g n
sudo n stable
sudo apt purge nodejs npm
exec $SHELL -l
node -v
npm -v
```

Execute npm install.
```
cd <project-dir>
npm install
```

## Build

```
npm run build
```

## Run

```
npm run serve
```
